# 🛡️ AVG Antivirus Website

A modern, responsive website for AVG Antivirus built with React, TypeScript, and Tailwind CSS.

![AVG Antivirus](https://via.placeholder.com/1200x600/16a34a/ffffff?text=AVG+Antivirus)

## ✨ Features

- **🚀 Modern Design** - Clean and professional UI with gradient backgrounds and animations
- **📱 Fully Responsive** - Works perfectly on all devices (mobile, tablet, desktop)
- **⚡ Fast Performance** - Optimized with React and lazy loading
- **🎨 Beautiful Animations** - Smooth transitions and hover effects
- **🔗 Support Integration** - All buttons linked to getchatsupport.live
- **📄 Multi-page Website** - Home, Features, Pricing, Support, About, Blog, Contact, Download, Login, Signup

## 📋 Pages Included

| Page | Description |
|------|-------------|
| **Home** | Hero section with trust badges, features grid, testimonials |

## 🛠️ Tech Stack

- **React 18** - Frontend library
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **React Router v6** - Routing
- **Headless UI** - Accessible components
- **Heroicons** - Icons
